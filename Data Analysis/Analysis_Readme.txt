# OpioidMortality_GTWR analysis
The R file in this folder contains the code to run the geographic and temporally weighted regression (GTWR) anlaysis, where mortality rate is the outcome, and opioids in grams, population size, percent of the population that is white, median age, and median household income are the independent variables. 

# "/Mortality Outcome GTWR Model Outputs 2006-2012.csv" 
Contains the data that was used to generate the model as well as the local coefficients for each county, generated by the model.
